# LinkCentral

LinkCentral is a Chrome extension that provides quick access to all your social media accounts with a single click. It allows you to add URLs for various platforms and open them directly from the extension. This tool is designed to enhance productivity and streamline your online experience.

## Features

- **Quick Access**: Access your social media accounts with just one click.
- **URL Management**: Easily add and save URLs for different platforms.
- **Responsive Design**: Clean and modern UI that adapts to different screen sizes.
- **Customizable**: Add URLs for various platforms including YouTube, Twitter, Facebook, GitHub, LinkedIn, Instagram, and more.

## Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/LinkCentral.git
    ```
2. **Navigate to the project directory:**
    ```bash
    cd LinkCentral
    ```
3. **Open Chrome and navigate to `chrome://extensions/`.**
4. **Enable Developer Mode by clicking the toggle switch in the top right corner.**
5. **Click the 'Load unpacked' button and select the project directory.**

## Usage

1. Click the LinkCentral icon in the Chrome toolbar.
2. For first-time users, click on an icon to enter and save your URL for that platform.
3. For returning users, simply click an icon to open your saved URL in a new tab.
4. To show or hide additional icons, click the arrow icon.
